Lists

A list is used to group related content in any component , where you can list items based on some order or no-order to it like un-ordered list.

Types of Lists:- Ordered List ,un-ordered List ,reverse order list , list with bullets, no-bullets and more other styles given to a lists.

Listing an items orderly using ol html tag, and un-ordered list using ul html tag. To reverse the order of ordered list, add reversed attribute to ol tag .
